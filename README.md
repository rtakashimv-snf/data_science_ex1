# Análise Exploratória de Dados – Índice Geral de Reclamações (IGR)
Este projeto realiza uma Análise Exploratória de Dados (EDA) utilizando um dataset público da Agência Nacional de Saúde Suplementar (ANS) Agência Nacional de Saúde Suplementar.

O conjunto de dados contém informações sobre o Índice Geral de Reclamações (IGR) das operadoras de planos de saúde no Brasil, considerando os meses de janeiro a outubro de 2022.

O IGR representa a média de reclamações para cada 10.000 beneficiários.

# Objetivo

O objetivo do trabalho foi:
- Analisar estatisticamente os valores do IGR
- Identificar padrões e dispersões
- Detectar possíveis outliers
- Comparar o comportamento do índice ao longo dos meses
